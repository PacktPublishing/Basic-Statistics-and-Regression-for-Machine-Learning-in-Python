Finding Means
-------------
Mean of x : Σx/n  = 201/18 = 11.167
Mean of x²: Σx²/n = 2969/18 = 164.944
Mean of y : Σy/n  = 1401/18 = 77.833


Finding Components
------------------
Sxx = Σx²- n(mean(x))² = 2969 - 18(11.167)² = 724.366

Sxy = Σxy - n(mean(x).mean(y)) = 16368 - 18(11.167*77.833) = 723.100

Sxx² = Σx³ - n(mean(x).mean(x²)) = 49701 - 18(11.167*164.944) = 16546.266

Sx²x² = Σx^4 - n(mean(x²).mean(x²)) = 892985 - 18(164.944*164.944) = 403267.584

Sx²y = Σx²y - n(mean(x²).mean(y)) = 255434 - 18(164.944*77.833) = 24348.446



Finding a, b, c
---------------

      [Sxy * Sx²x²] - [Sx²y * Sxx²]
b =    ----------------------------
	  [Sxx * Sx²x²] - [(Sxx²)²]
      
    = [723.100 * 403267.584] - [24348.446 * 16546.266]
       ----------------------------------------------
      [724.366 * 403267.584] - [(16546.266)²]
    
    = 2916,02,789.9904 - 4028,75,864.202636
       -----------------------------------
      2921,13,326.751744 - 2737,78,918.542756
      
    = -1112,73,074.212236 / 183,34,408.208988 = -6.0690
      
      
      
      [Sx²y * Sxx] - [Sxy * Sxx²]
c =    ----------------------------
	  [Sxx * Sx²x²] - [(Sxx²)²]
      
    =  [24348.446 * 724.366] - [723.100 * 16546.266]
        --------------------------------------------
        [724.366 * 403267.584] - [(16546.266)²]
        
    =  176,37,186.435236 - 119,64,604.9446
       -----------------------------------
       2921,13,326.751744 - 2737,78,918.542756
    
    = 56,72,581.490636 / 183,34,408.208988 = 0.3094
      


a =   mean(y) - b(mean(x)) - c(mean(x²)) 
    = 77.833 - (-6.0690)*(11.167) - 0.3094*164.944
    = 77.833 + 67.772 - 51.034
    = 94.571
 


y = a+bx+cx²
------------ 

y = 94.571-6.0690x+0.3094x²

if x = 20
y = 94.571-6.0690*20+0.3094*20²
  = 94.571 - 121.38 + 123.76
  = 96.951

corelation coefficient for polynomial regression
------------------------------------------------
r² = 1 - Σ(y -(a+bx+cx²))² / Σ(y−¯𝑦)²
   = 1 - 1032.911588 / 4170.500002
   = 1- 0.24767
   = 0.75233
